# Code of Conduct

Be decent. This is a small project — assume good faith, keep it constructive, and
don't be a dick. Harassment, demeaning comments, and personal attacks aren't welcome
in issues, PRs, or anywhere else in the project.

Report problems to the maintainer at the email on the GitHub profile. Maintainers may
edit, hide, or remove contributions that violate this, and may block repeat offenders.

Adapted in spirit from the [Contributor Covenant](https://www.contributor-covenant.org).
